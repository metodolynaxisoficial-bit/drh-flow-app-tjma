Crie o sistema DRH Flow em formato de aplicativo multiplataforma, com interface e experiência adaptadas para:

- iOS
- Android
- Web responsiva

O DRH Flow é um sistema inteligente de atendimento e gestão funcional do Tribunal de Justiça do Maranhão, voltado para:

- servidores
- estagiários
- residentes
- aposentados

O aplicativo deve ter aparência institucional, moderna, limpa, tecnológica e funcional, com alto nível de organização visual e excelente usabilidade em celular e desktop.

OBJETIVO DO SISTEMA:

Centralizar, organizar e automatizar os serviços de Recursos Humanos em um único ambiente digital, facilitando o acesso a informações funcionais, solicitações administrativas, orientações sobre direitos e deveres, acompanhamento de demandas e atendimento automatizado com inteligência artificial.

O sistema deve funcionar como app nativo para iOS e Android e também como plataforma web responsiva, com adaptação automática de layout, navegação e componentes conforme o dispositivo.

ESTRUTURA GERAL DO APP:

Criar o DRH Flow como um sistema multiplataforma com as seguintes áreas:

1. Tela inicial / home
2. Login
3. Cadastro inicial
4. Painel principal do usuário
5. Área de solicitações
6. Área de teletrabalho
7. Área de direitos e benefícios
8. Área de acompanhamento funcional
9. Atendimento via WhatsApp
10. Assistente virtual integrada
11. Notificações
12. Perfil do usuário
13. Configurações
14. Área administrativa / gestão interna

TELA INICIAL:

A tela inicial deve apresentar:

- nome do sistema: DRH Flow
- subtítulo: Sistema Inteligente de Gestão Funcional
- breve texto institucional explicando o sistema
- botão de acesso ao atendimento
- botão de login
- destaque para integração com WhatsApp
- destaque para assistente virtual
- visual institucional com sensação de inovação, organização, eficiência e confiança

LOGIN E CADASTRO:

Criar telas para:

- login com matrícula e senha
- recuperação de senha
- primeiro acesso
- autenticação segura
- validação por CPF, matrícula ou dado funcional
- cadastro com fluxo claro e intuitivo

PAINEL PRINCIPAL:

O painel principal deve exibir, de forma organizada:

- atalhos rápidos
- andamento de solicitações
- notificações
- serviços mais acessados
- acesso à área de teletrabalho
- acesso ao atendimento via WhatsApp
- acesso à assistente virtual
- status de demandas funcionais

ÁREA DE TELETRABALHO:

Criar uma seção específica sobre teletrabalho, com:

- explicação institucional
- critérios de elegibilidade
- análise individual de elegibilidade
- condições especiais de trabalho
- orientação sobre teletrabalho para servidores com deficiência ou dependentes com deficiência
- destaque para atualização normativa
- possibilidade de consulta automatizada
- linguagem clara, didática e institucional

A seção deve considerar, conceitualmente:

- Resolução CNJ nº 227/2016 e alterações posteriores
- Resolução CNJ nº 343/2020
- Instrução Normativa CNJ nº 98/2024
- Resolução GP nº 99/2020 do TJMA
- Resolução GP nº 88/2022 do TJMA
- Resolução GP nº 91/2020 do TJMA

ÁREA DE DIREITOS E BENEFÍCIOS:

Criar uma área com:

- orientações sobre direitos funcionais
- benefícios
- licenças
- aposentadoria
- jornada especial
- teletrabalho
- consultas rápidas
- conteúdo organizado por categorias

ÁREA DE SOLICITAÇÕES:

Permitir estrutura para:

- abertura de solicitações
- envio de documentos
- acompanhamento de status
- histórico de pedidos
- mensagens automáticas
- andamento do processo interno

WHATSAPP INTEGRADO:

O sistema deve prever integração com WhatsApp institucional, com:

- botão direto “Falar no WhatsApp”
- continuidade do atendimento iniciado no sistema
- respostas automáticas baseadas nas informações do DRH Flow
- envio de orientações, status de solicitações e dúvidas frequentes
- experiência fluida entre app, web e WhatsApp

ASSISTENTE VIRTUAL:

O sistema também deve conter uma assistente virtual integrada.

Para fins de apresentação do projeto, essa assistente será denominada Artemis, com avatar humano ilustrativo e institucional.

Deve ficar claro no design e no texto que:

- Artemis é um avatar demonstrativo para apresentação do projeto
- a identidade final da assistente pode ser adaptada pelo Tribunal
- a assistente oficial, como ANA, pode ser integrada futuramente
- a assistente atua no sistema e também via WhatsApp

A assistente virtual deve:

- responder dúvidas
- orientar navegação
- apresentar informações funcionais
- apoiar o usuário em teletrabalho, solicitações e direitos
- ter linguagem clara, objetiva e institucional

ÁREA ADMINISTRATIVA:

Criar estrutura para área interna de gestão, contendo:

- visualização de demandas
- acompanhamento de solicitações
- organização por filas ou categorias
- painel gerencial
- rastreamento de atendimentos
- acompanhamento de interações do sistema e WhatsApp
- análise de dados e indicadores de uso

DESIGN E EXPERIÊNCIA:

O design deve ser:

- institucional
- elegante
- tecnológico
- moderno
- minimalista
- altamente organizado
- acessível
- responsivo

Para mobile:
- navegação simples
- ícones claros
- bottom navigation ou menu funcional
- leitura confortável
- botões bem posicionados

Para web:
- dashboard mais amplo
- menus laterais
- cards organizados
- estrutura executiva e produtiva
- aproveitamento inteligente do espaço

CRIAR TAMBÉM:

- ícones próprios do sistema
- widgets ou cards informativos
- blocos de acesso rápido
- componentes reutilizáveis
- identidade visual consistente entre iOS, Android e Web

BENEFÍCIOS A DESTACAR:

- redução de burocracia
- maior eficiência administrativa
- centralização do atendimento
- transparência nas demandas
- melhor experiência do usuário
- automatização de processos
- integração digital com WhatsApp
- suporte inteligente com assistente virtual

IMPORTANTE:

O Figma deve gerar esse sistema já como produto digital real, em formato de aplicativo multiplataforma, e não apenas como uma peça conceitual estática.

Criar as telas com lógica de navegação entre elas, estrutura coerente de fluxo e componentes reutilizáveis, mantendo consistência visual e funcional em todas as versões.

O tom geral do produto deve transmitir:

- confiança institucional
- inovação
- eficiência
- clareza
- modernização do atendimento do Tribunal de Justiça do Maranhão
