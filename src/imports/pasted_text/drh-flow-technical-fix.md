## CORREÇÕES PERMITIDAS (SOMENTE ESTAS — NADA ALÉM)

### 1. Renomear arquivos e pastas para minúsculo

- Renomear para lowercase (ex: `Dashboard.tsx` → `dashboard.tsx`)
- **OBRIGATÓRIO:** ao renomear qualquer arquivo, atualize TODOS os imports que o referenciam em TODO o projeto
- **OBRIGATÓRIO:** valide com busca global antes de finalizar (grep/find)
- Componentes React internos continuam em PascalCase no código (`export default function Dashboard`)
- Apenas o NOME DO ARQUIVO muda, não o nome do componente

### 2. Corrigir caminhos de import (case-sensitive)

- Todos os caminhos de import devem corresponder EXATAMENTE ao nome do arquivo em disco
- Se um import referencia um arquivo que NÃO EXISTE:
  - Comente o import com `// TODO: arquivo não encontrado — verificar manualmente`
  - NÃO crie o arquivo
  - NÃO substitua por outro componente

### 3. Remover dependências específicas do Figma

Remova SOMENTE estes pacotes (se existirem):

- `@figma/plugin-typings`
- `@figma/rest-api-spec`
- `@figma/widget-typings`
- Qualquer pacote que comece com `@figma/`
- Qualquer import que comece com `figma:`

**ATENÇÃO CRÍTICA:** Se um componente visual usa um wrapper gerado pelo Figma (ex: `FigmaFrame`, `AutoLayout`):

- Substitua SOMENTE o wrapper, mantendo o JSX interno 100% idêntico
- Se não souber como substituir sem alterar o visual, NÃO mexa — comente com `// TODO: wrapper Figma — substituir manualmente`

### 4. Corrigir bloco scripts do package.json

Altere SOMENTE o bloco `"scripts"`:

```json
"scripts": {
  "dev": "vite",
  "build": "vite build",
  "preview": "vite preview"
}
```

**NÃO toque em:** `dependencies`, `devDependencies`, `name`, `version`, `type`, ou qualquer outro campo.

### 5. Corrigir imports de ícones

- Biblioteca padrão do projeto: **Lucide React** (`lucide-react`)
- Se o projeto usa outra biblioteca de ícones, mantenha a que já está instalada
- Corrija apenas nomes de ícones que não existem na biblioteca (ex: `IconeDashboard` → verificar nome correto em lucide-react)
- Se não encontrar o ícone correto, comente o import com `// TODO: ícone não encontrado`
